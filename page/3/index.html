еҸ‘иЎЁдәҺ 2020-06-28 еҲҶзұ»дәҺ algorithm

жҰӮеҝө

дәҢеҲҶжі•жҳҜз”ЁеңЁжңүеәҸзҡ„ж•°жҚ®дёӯеҝ«йҖҹзӯӣйҖүж•°жҚ®зҡ„ж–№жі•гҖӮ

жіЁж„ҸзӮ№

е…ідәҺдёӢз•ҢLе’ҢдёҠз•ҢRпјҢеҜ№дәҺж•°з»„жқҘиҜҙпјҢеҒҮи®ҫжҲ‘д»¬ж–°е»әдёҖдёӘй•ҝеәҰдёәNзҡ„ж•°з»„пјҢз”Ёж•°еӯҰдёҠзҡ„е®ҡд№үзҡ„жқҘиҜҙпјҢжҳҜ[0,N)пјҢе·Ұй—ӯеҸіејҖзҡ„еҢәй—ҙгҖӮеҶіе®ҡеҢәй—ҙејҖй—ӯзҡ„еҗҢж—¶пјҢд№ҹеҶіе®ҡдәҶжҗңзҙўзҡ„еҢәй—ҙе’ҢдәҢеҲҶзҡ„з»ҲжӯўжқЎд»¶пјҲжІЎжңүеҸҜд»ҘиҝӣиЎҢжҗңзҙўзҡ„еҢәй—ҙпјүпјҢжңҖзӣҙжҺҘзҡ„е°ұжҳҜеҸҚеә”еңЁleftе’ҢrightеңЁеҫӘзҺҜиҝҮзЁӢдёӯзҡ„еҸҳеҢ–гҖӮиҜҰз»ҶзңӢдёӢйқўгҖӮ

code & analysis

дёҖгҖҒж ҮеҮҶдәҢеҲҶ

ж ҮеҮҶдәҢеҲҶжҳҜе·Ұй—ӯеҸій—ӯзҡ„еҢәй—ҙпјҢжҗңзҙўеҢәй—ҙдёә[left,right]пјҢжіЁж„ҸдәҢеҲҶз»Ҳжӯўзҡ„жқЎд»¶жҳҜ left <= rightпјҢеӣ дёәз»ҲжӯўдәҢеҲҶзҡ„жқЎд»¶жҳҜжІЎжңүеҸҜд»ҘиҝӣиЎҢжҗңзҙўзҡ„еҢәй—ҙдәҶпјҢеҪ“жҗңзҙўеҢәй—ҙдёә[right+1,right]пјҢжҗңзҙўеҢәй—ҙдёәз©әпјҢз»ҲжӯўжқЎд»¶жӯЈзЎ®гҖӮеҰӮжһңеҶҷжҲҗ left < right пјҢе°ұдјҡеҮәзҺ°еҪ“left=2пјҢright=2пјҢеҲҷжҗңзҙўеҢәй—ҙдёә[2,2]пјҢжӯӨж—¶жҗңзҙўеҢәй—ҙжҲҗз«ӢпјҢиҝҳйңҖиҰҒиҝӣиЎҢжҗңзҙўпјҢдҪҶдәҢеҲҶжҗңзҙўе·Із»Ҹз»ҲжӯўпјҢз»ҲжӯўжқЎд»¶й”ҷиҜҜгҖӮ

дәҢгҖҒе·Ұй—ӯеҸіејҖзҡ„дәҢеҲҶ

еҗҢж ҮеҮҶдәҢеҲҶдёҖж ·еҲҶжһҗпјҢе·Ұй—ӯеҸіејҖзҡ„дәҢеҲҶпјҢеҲҷжҗңзҙўеҢәй—ҙжҳҜ[left,right)пјҢдәҢеҲҶзҡ„з»ҲжӯўжқЎд»¶жҳҜ left < right пјҢеҪ“жҗңзҙўеҢәй—ҙдёә[right,right)пјҢжҗңзҙўеҢәй—ҙдёәз©әпјҢз»ҲжӯўжқЎд»¶жӯЈзЎ®гҖӮжіЁж„Ҹright = midзҡ„ж“ҚдҪңпјҢиҖҢдёҚжҳҜright = mid - 1пјҢеӣ дёәrightеҲқе§ӢеҖјдёәnums.lengthпјҢиҝҷдёӘзҙўеј•дҪҚзҪ®жҳҜдёҚиғҪи®ҝй—®зҡ„пјҢеҸҰдёҖдёӘеҺҹеӣ жҳҜеҸіејҖпјҢrrightиҰҒе§Ӣз»ҲйҒөе®ҲиөӢеҖјдёәеҸ–дёҚеҲ°зҡ„еҸіиҫ№з•ҢгҖӮ

дёүгҖҒйҮҚеӨҚе…ғзҙ зҡ„е·Ұз•ҢпјҲдёӢз•ҢпјүжҗңзҙўдәҢеҲҶ

е·Ұз•ҢдәҢеҲҶжҳҜеҪ“еҮәзҺ°йҮҚеӨҚе…ғзҙ зҡ„ж—¶еҖҷпјҢжүҫеҲ°жңҖе·ҰеҖјзҡ„зҙўеј•гҖӮиҝҷйҮҢдёҚеҲҶжһҗиҫ№з•Ңзҡ„жғ…еҶөпјҢиҝҷйҮҢдё»иҰҒиҜҙжҳҺжүҫжңҖе·ҰеҖјзҡ„ж—¶еә”иҜҘзҹҘйҒ“зҡ„зҹҘиҜҶгҖӮеңЁnums[mid] == targetж—¶пјҢе°Ҷright = midпјҢиҖҢдёҚжҳҜreturnпјҢжҳҜдёәдәҶз»§з»ӯеҫҖе·Ұз•ҢжҗңзҙўгҖӮжңҖз»Ҳеҫ—еҲ°leftиЎЁзӨәжҜ”targetе°Ҹзҡ„ж•°жңүleftдёӘпјҢжіЁж„ҸleftеҸҜиғҪдёәnums.lengthпјҢеӣ дёәдёҖзӣҙжҗңдёҚеҲ°пјҢжңҖеҗҺmid + 1еҫ—еҲ°nums.lengthгҖӮеӣ жӯӨпјҢж №жҚ®д№ӢеүҚдәҢеҲҶжҗңдёҚеҲ°зҡ„жғ…еҶөдёӢиҝ”еӣһ-1пјҢжңҖеҗҺзЁӢеәҸеҚ•зӢ¬еӨ„зҗҶдёҖдёӢеҚіеҸҜгҖӮ

еӣӣгҖҒйҮҚеӨҚе…ғзҙ зҡ„еҸіз•ҢпјҲдёҠз•ҢпјүжҗңзҙўдәҢеҲҶ

жәўеҮәеӨ„зҗҶ

L + RжәўеҮәзҡ„жғ…еҶөпјҢеҸҜд»ҘиҝӣиЎҢиҪ¬жҚўпјҡпјҲL + Rпјү/ 2 = пјҲ2L + R-Lпјү/ 2 = L + пјҲR - Lпјү/ 2пјҢд»Јз ҒдёӯеҶҷжҲҗ L + ((R - L) >> 1)гҖӮ

жңүдәӣж—¶еҖҷдёҠйқўдёҖжӯҘзҡ„иҪ¬жҚўе№¶дёҚиғҪйҒҝе…ҚжәўеҮәзҡ„й—®йўҳпјҢиҝҷжҳҜеӣ дёәж•°жҚ®иҢғеӣҙзҡ„й—®йўҳпјҢж— жі•з”Ёж•°еӯҰзҡ„ж–№жі•йҒҝе…ҚпјҢдҫӢеҰӮintжғ…еҶөдёӢпјҢдҪҝз”ЁL + ((R - L) >> 1)иҝҳжҳҜж— жі•йҒҝе…ҚжәўеҮәжҳҜintж•°жҚ®иҢғеӣҙеҜјиҮҙзҡ„и®Ўз®—иҝҮзЁӢж— жі•и§„йҒҝжәўеҮәй—®йўҳпјҢжӯӨж—¶еә”иҜҘдҪҝз”ЁжӣҙеӨ§иҢғеӣҙзҡ„ж•°жҚ®зұ»еһӢиҝӣиЎҢеӨ„зҗҶпјҢдҫӢеҰӮе…Ҳз”ЁlongиҝӣиЎҢи®Ўз®—пјҢжңҖеҗҺreturnзҡ„ж—¶еҖҷејәиҪ¬еӣһintгҖӮ
